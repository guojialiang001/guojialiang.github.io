# Radio

单选选择组件

# 基础用法




设置v-model绑定变量, 绑定变量如果和相应Radio的val的值相等，那么即为选中，val可以为 String Number Boolean类型。



 
 






 







<template>
  <div class="demo">
     <myca-radio  v-model="radio2" val="1" >选项1</myca-radio>   
     <myca-radio  v-model="radio2" val="2" >选项2</myca-radio>    
  </div>
</template>
<script>
export default {
  data() {
    return {
      radio2: "2"
    };
  },
  methods: {
  }
};
</script>

# 单选框组

单选框以组的形式排列

提示

以myca-radio-group组绑定v-model，在myca-radio中设置好val即可，无需再给每一个myca-radio绑定变量


 
 
 
 





 





 <template>
  <myca-radio-group  v-model="radio">
        <myca-radio   val="西瓜" >西瓜</myca-radio>   
        <myca-radio   val="橘子" >橘子</myca-radio>
  </myca-radio-group> 
</template>
<script>
export default {
  data() {
    return {
      radio: "西瓜",
    };
  },
};
</script>

# Radio属性列表

参数 说明 类型 可选值 默认值
v-model 绑定值 string / number / boolean
val RadioGroup中使用的value值 string / number / boolean
disabled 是否禁用 boolean false
size Radio 的尺寸 string medium / small / mini medium

# Radio事件列表

事件名称 说明 回调参数
change 绑定值变化时触发的事件 选中的Radio的val值

# RadioGroup属性列表

参数 说明 类型 可选值 默认值
v-model 绑定值 string / number / boolean
disabled 是否禁用 boolean false
size Radio 的尺寸 string medium / small / mini medium
color Radio的颜色 string medium / small / mini medium

# RadioGroup事件列表

事件名称 说明 回调参数
change 绑定值变化时触发的事件 选中的 Radio的val值
