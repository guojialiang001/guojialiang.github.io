# 安装

基于vue2.x的UI框架,想要快速上手,就需要我们快速安装并使用。

# npm安装

npm i myrcia-test -S

# CDN


 

 

<!-- 引入样式 -->
<link rel="stylesheet" href="https://unpkg.com/myca-ui/lib/myca-ui.css">
<!-- 引入组件库 -->
<script src="https://unpkg.com/myca-ui/lib/myca-ui.umd.min.js"></script>

# 全局引入

全局引入,将所有的组件都引入到项目中,就可以随意使用该组件库下的所有组件了。




 
 
 






import Vue from 'vue'
import App from './App'

import mycaUi from 'myca-ui'
import 'myca-ui/lib/myca-ui.css';
Vue.use(mycaUi)

new Vue({
  el: '#app',
  render: h => h(App)
});

# 按需引入

借助babel-plugin-import我们可以只引入需要的组件，以达到减小项目体积的目的。
首先，安装 babel-plugin-import：
npm install babel-plugin-import --save-dev
然后，将 .babelrc 修改为：





 
 
 
 



// .babelrc
{
   //上略
  "plugins": [
    ["import", {
    "libraryName": "myca-ui",
    "libraryDirectory": "packages"}
    ]
  ]
}

接下来，如果你只希望引入部分组件，比如 Radio 和 Test，那么需要在 main.js 中写入以下内容：




 
 
 






import Vue from 'vue'
import App from './App'

import { Radio,Test } from 'myca-ui'
Vue.component(Radio.name, Radio);
Vue.component(Test.name, Test);

new Vue({
  el: '#app',
  render: h => h(App)
});
